# Anime Contact Explorer (SwiftUI)

This is a professional and interactive contact management application built with **SwiftUI**. The project demonstrates core iOS development concepts, including data modeling, list rendering, and seamless navigation.

## 🚀 Key Features

*   **Dynamic List View:** Displays a collection of characters with real-time status and contact snippets.
*   **Master-Detail Navigation:** Implements `NavigationStack` for smooth transitions between the main list and detailed profile views.
*   **Custom UI Design:** Features high-quality image rendering with circular clipping, stroke overlays, and organized info sections using `List` and `Section`.
*   **Interactive Data Management:** 
    *   **Swipe to Delete:** Easily remove contacts using `.onDelete`.
    *   **Rearrange:** Reorder contacts using the `.onMove` modifier and `EditButton`.
*   **Modern Iconography:** Uses Apple's **SF Symbols** for a clean and native look.

## 🛠️ Technical Stack

*   **Language:** Swift 5.0+
*   **Framework:** SwiftUI
*   **Architecture:** Model-driven UI using the `Identifiable` protocol.
